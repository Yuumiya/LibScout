The Problem

For Humans

Niche libraries often lack real-world examples. "Hello World" isn't enough for production code.

For AI (LLMs)

Knowledge cutoffs make AI blind to new releases. AI hallucinates APIs for libraries it doesn't know.

Architecture

Stack: Python (FastAPI), ChromaDB, Next.js, MCP SDK.

The Secret Sauce: Tree-sitter

  • We don't use "dumb" Regex search.
  • We parse the Concrete Syntax Tree (CST) to extract full functional contexts.

;; Sample Tree-sitter Query (Lisp)
(call_expression
  function: (member_expression
    object: (identifier) @lib_name (#eq? @lib_name "my-lib")
    property: (identifier) @method))

Final Project

A dual-interface search engine:

  1. Web UI: Semantic search for humans (Search by intent, not just keyword).
  2. MCP Server: A standardised tool for AI Agents (Claude, Cursor) to "look up" library usage.
MUST HAVE: GitHub Crawler, Tree-sitter Parser, Vector Index, MCP Endpoint.
